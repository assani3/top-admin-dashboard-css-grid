# Admin Dashboard

A modern administrative dashboard built with **HTML5**, **CSS3**, and **CSS Grid** as part of **The Odin Project Intermediate HTML & CSS Curriculum**.

## Overview

This project focuses on mastering **CSS Grid Layout** by recreating a professional dashboard interface. The application demonstrates advanced layout techniques, nested grid structures, component organization, and modern UI design principles.

The dashboard includes:

* Responsive sidebar navigation
* Top header with search functionality
* User profile section
* Action buttons
* Project cards
* Announcements panel
* Trending section
* Modern dashboard styling

The primary objective was to gain hands-on experience with CSS Grid by creating complex page layouts using nested grids and organized content sections.

## Installation

1. Clone the repository

```bash
git clone https://github.com/assani3/top-admin-dashboard-css-grid.git
```

2. Navigate into the project folder

```bash
cd admin-dashboard
```

3. Open `index.html` in your browser

or

Use VS Code Live Server:

```bash
Right Click index.html
→ Open with Live Server
```
